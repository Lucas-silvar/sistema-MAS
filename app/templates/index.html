Sobre o Sistema de Análise de Microplásticos

Este sistema é resultado do projeto de iniciação científica intitulado “Desenvolvimento de um software para quantificação e análise de microplásticos”, desenvolvido no laboratório LCTM da UTFPR, Campus Pato Branco, PR, Brasil. Concebido para atender às demandas de pesquisadores e estudantes, o sistema oferece uma solução robusta para a análise automatizada de microplásticos a partir de imagens digitais.

O objetivo central do projeto é fornecer uma ferramenta computacional que otimize a análise de amostras, permitindo a contagem, medição e determinação da razão de aspecto de partículas de microplásticos de forma rápida e precisa. Dessa forma, busca-se acelerar o processamento de dados experimentais, garantindo a integridade e a rastreabilidade dos resultados obtidos.
